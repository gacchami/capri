このサイトについて
　ようこそおいで下さいました。このサイトは、鉄道情景写真をメインとしてその他さまざまな写真を公開する個人サイトです。
当サイト内の各著作物はバナーを除き全て管理人に著作権がございます。壁紙素材も含め無断転載・再配布は一切禁止です。
　動作環境
　当サイトは、以下の環境で動作を確認しております。
　　OS 　　 ：Mac OS 10.4.11 / Windows XP Home Edition
　　ブラウザ：Safari 3.0 / firefox2 / Internet Explorer 6
　　解像度　：1280*1024 / 1024*768
　掲示板について
　画像投稿が出来る掲示板です。最低限のマナーを必ず守った上、お気軽にご投稿下さい。不適切だと思われる書き込みは予告なしに削除させて頂きます。
　リンクについて
　リンクフリーです。ただしトップページのみ。相互リンクをご希望の方は、メールにてサイト名、管理者HN、アドレス等を明記し、お気軽にご一報下さい。相互リンクを受付させて頂きます。
　但し、管理人の都合上相互リンクをお引き受けできない場合がございますのでご了承下さい。

　選べる三タイプ。相互リンクの際にご使用ください。サーバーの都合上、バナーはダウンロードして頂く様お願い致します。
